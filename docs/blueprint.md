# **App Name**: Landcraft Portfolio

## Core Features:

- Homepage Showcase: Visually appealing showcase of Landcraft Holdings' key projects.
- About Us: Information about Landcraft Holdings, its vision, and team.
- Project Galleries: Dedicated pages for each project (Uttara, Aftabnagar, Banani, Mirpur) featuring stunning visuals.
- Contact Form: A simple form for inquiries, linked to Landcraft's email.
- SEO Optimization: Implementation of SEO best practices with real estate keywords relevant to Dhaka, Bangladesh.
- Image Enhancement Tool: Use AI to automatically enhance and optimize images of properties before displaying them on the website, ensuring high visual quality using a 'tool' for enhancing image resolution, adjusting brightness/contrast, and optimizing for web.

## Style Guidelines:

- Primary color: Deep Mustard Yellow (#D4A00D), drawing inspiration from the brand logo.
- Background color: Desaturated Mustard (#FAF0D7) for a clean, bright canvas.
- Accent color: Analogous, muted Orange (#D46A0D) for highlights and call to actions.
- Body font: 'PT Sans', a humanist sans-serif font for body text and legibility.
- Headline font: 'Playfair', a modern serif font for elegant and fashionable titles. Paired with PT Sans, provides better legibility for content blocks.
- Clean, modern icons to represent different sections and features, complementing the overall design. Will include an edited version of the logo for favicon.
- Modern, grid-based layout to showcase projects effectively. Focus on high-quality images and visuals.
- Subtle animations and transitions for a smooth and engaging user experience.
